# Disprog UAS

Gatau wes

## Instalasi
```
git clone https://github.com/irevle/disprog
```
Lalu dalam NetBeans,
```
1. Open Project
2. Buka ClientTCP dan ServerTCP
```
## Cara Update Data

Sebelum melakukan perubahan, lakukan
```
git pull
```
untuk update kode dari GitHub ke lokal

## Mendorong perubahan dari lokal ke GitHub
```
git add .
git commit -am "[Pesan perubahan mu disini]"
git push
```
